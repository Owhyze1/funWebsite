Browser Object Model

Browser Object Model (BOM) represents the current browser window.

Main Objects available inside the window

  • document
  • history
  • location
  • screen
  • navigator

The document object

The document object represents web page's structure and content in a tree format, where the tree nodes represent HTML elements and text content.

The history object

Holds data about the URL's that have been visited

The location object

Used to access the URL of the document currently loaded in the window and also provides information about the general navigation functionality of the document

The screen object

Holds info about screen like width and height

The navigator object

Contains information about user's browser


To access browser object model, use the window object window.document.getElementById('topbar')

OR you can simply use document.getElementById('topbar') because all the global javascipt objects, functions, and variables

automatically bec0me members of the window object.

Width of Window

window.innerWidth

Height of Window

window.innerHeight

Other Window Methods

  • setTimeout( ) - performs action after specified amount of time
  • setInterval( ) - performs action after a set interval of time
  • alert( ) - display alert box
  • confirm( ) - display confirm dialog box with "OK" and "Cancel" buttons
  • prompt( ) - displays dialog box to get input from user
  • open( ) - opens an entered URL in a new tab
  • close( ) - closes the current window

Screen and Navigator objects

The screen object

Holds informaiton about the browser screen

  • screen width
  • screen height
  • availableWidth
  • availableHeight
  • colorDepth
  • pixelDepth

To access the screen object, you can do window.screen or you can simply write screen and it returns the same properties.

This returns the following example object

{
availWidth: 1536,
availHeight: 824,
width: 1536,
height: 864,
colorDepth: 24,
...
}

The navigator object

Contains information about the browser where the application is running

You can access it by using window.navigator

You can use the user agent string to get informaiton about the operating system and the browser.

Access it with navigator.userAgent

Take the output and put it into a user agent parser, found on Google, to decipher the information.

Use the navigator.onLine property to see if a user has an active internet connection (returns true).

The history object

An array object of URLs previously visited by the user

    Two Functions
  • back( )
  • forward( )

These functions allow you to go to the previous or next page

Calling this function window.history.back() will load the previous page

The location object

Used to get details of current page address and can redirect browser to new page in same window.

Use window.location to get a Location object

window.location.href is the current web addres

window.location.hostname returns the host name (i.e. www.google.com)

window.location.pathname shows the path after the host name

window.location.search returns a search string (anything after ?)

window.location.assign('www.google.com') loads the website into the browser
