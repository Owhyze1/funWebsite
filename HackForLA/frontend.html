Process for Installing and Running Home Unite Us Calendaring Service

Steps

  1. Fork Repo to Owhyze1 github
  2. Create hfla directory in C:/Coding
  3. git clone to hfla folder
  4. Install Flask
    • In project folder --> py -3 -m venv venv
    • venv\Scripts\activate
    • pip install flask
    • Upgrade flask --> python.exe -m pip install --upgrade pip
  5. Run Flask application
    • Error --> No Module named flask
    • Solution
    • Open new command window and use command pip show flask to see where flask is installed
    • New error --> WARNING: Package(s) now found: Flask
    • Use pip install flask
    • now pip show flask works
  6. Retry on Linux
    • Follow instructions above for installing flask on linux
    • flask run starts but...
    • ERROR --> ModuleNotFoundError: No module named 'bson'
  7. FINAL SOLUTION
    • URL: VS Code Docs
    • Navigate to HomeUniteUs project folder and type the following into linux command line
    • pip3 install -r requirements.txt
